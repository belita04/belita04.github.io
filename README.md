# 🐶 이명진의 포트폴리오 페이지

안녕하세요!

## ✨ 주요 기능

- 🌟 **네비게이션**: 동물 이모지로 구성된 네비게이션 메뉴
- 🖼️ **GIF 프로필 이미지**: 움직이는 프로필 이미지
- 📝 **섹션별 구성**: 소개, 경력, 학력, 전공, 프로젝트 등으로 구성된 포트폴리오
- 📱 **연락처 & 채널 링크**: 이메일, 전화번호, GitHub, 블로그 등 다양한 채널 제공
- 📌 **고정된 푸터**: 화면 하단에 고정되어있는 저작권 및 연락처 정보
